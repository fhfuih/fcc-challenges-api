URL Shortener Service üîóüí•

POST <host>/api/shorturl/new - application/json { "url": "Valid HTTP(S) URL" }

This will resond you with both the original and the shortened URL.

GET <host>/api/shorturl/<id>

This will redirect you to the corresponding website.

Note that the database is capped to 4096MB. Old records are not guaranteed to perserve forever.

This shorturl service is for demo only. I am not responsible for any URL entries stored in this database, nor any content the URLs point to.

Input 
<project>/api/shorturl/new - { "url": "https://github.com" }
Output 
{
    "url": "https://github.com",
    "short": "wQLKLM4Cc"
}
Input 
<project>/api/shorturl/wQLKLM4Cc
Output 
REDIRECT - https://github.com
