URL Shortener Service üîóüí•

POST <host>/api/shorturl/new
x-www-form-urlencoded { url }

This will respond you with both the original and the shortened URL, if the URL is of HTTP(S) protocol whose host passes dns.lookup.

GET <host>/api/shorturl/<id>

This will redirect you to the corresponding website.

GET <host>/api/shorturl

[Debug] This will respond you with all short URL records stored.

Input 
<project>/api/shorturl/new - { "url": "https://github.com" }
Output 
{
    "url": "https://github.com",
    "short": "wQLKLM4Cc"
}
Input 
<project>/api/shorturl/wQLKLM4Cc
Output 
REDIRECT - https://github.com

Exercise Tracker üí¶üí™

POST <host>/api/exercise/user/new
x-www-form-urlencoded { username }

This will create a user and respond you with the user id and username.

GET <host>/api/exercise/user/:username

This will respond you with the specified user id and username, or a NotFoundError.

GET <host>/api/exercise/user

[Debug] This will respond you with all the user records stored.

POST <host>/api/exercise/new
x-www-form-urlencoded { userId, description, duration (int), date (yyyy-mm-dd) }

This will create an exercise record and respond you with the created exercise document.

GET <host>/api/exercise/log?{userId}[&from(yyyy-mm-dd)][&to(yyyy-mm-dd)][&limit(+int)]

This will list all exercise records belonged to a specified user in date ascending order with optional filters.

Create User 
POST <project>/api/exercise/user/new
Create Exercise 
POST <project>/api/exercise/new
List Exercise 
GET <project>/api/exercise/log
